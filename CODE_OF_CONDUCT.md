# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) version 2.1 as its code of conduct.

The full text is available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## Enforcement

Reports about conduct in this project's spaces can be sent privately via [GitHub Security Advisories](https://github.com/QuentinDecobert/meridian/security/advisories/new) or by contacting the maintainer through their [GitHub profile](https://github.com/QuentinDecobert).

All reports will be reviewed and investigated promptly and fairly. Confidentiality will be maintained with regard to the reporter of an incident.
